# 🧠 Logic Box - Pattern Generator & Number Analyzer

## 📌 Description
Logic Box is a beginner Python project that combines pattern generation and number analysis into a menu-driven program.

---

## 🚀 Features
- ⭐ Star pattern generator
- 🔢 Even/odd number detection
- ➕ Sum calculation of range
- 🔁 Menu-driven program using loops
- ❌ Exit option

---

## 🛠️ Technologies Used
- Python (Core Concepts)
  - Loops
  - Conditionals
  - User Input
  - Nested loops
